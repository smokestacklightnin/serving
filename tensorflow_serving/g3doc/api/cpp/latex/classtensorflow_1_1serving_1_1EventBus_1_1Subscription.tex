\subsubsection*{template$<$typename E$>$\newline
class tensorflow\+::serving\+::\+Event\+Bus$<$ E $>$\+::\+Subscription}

\mbox{\hyperlink{classtensorflow_1_1serving_1_1EventBus_1_1Subscription}{Subscription}} is an RAII object and tracks the lifecycle of a single subscription to an \mbox{\hyperlink{classtensorflow_1_1serving_1_1EventBus}{Event\+Bus}}. Upon destruction, it automatically Unsubscribes from the originating \mbox{\hyperlink{classtensorflow_1_1serving_1_1EventBus}{Event\+Bus}}.

Note that \mbox{\hyperlink{classtensorflow_1_1serving_1_1EventBus_1_1Subscription}{Subscription}} only maintains weak\+\_\+ptr references to the \mbox{\hyperlink{classtensorflow_1_1serving_1_1EventBus}{Event\+Bus}}, such that the \mbox{\hyperlink{classtensorflow_1_1serving_1_1EventBus}{Event\+Bus}} and Subscriptions can be safely destructed in any order. \mbox{\hyperlink{classtensorflow_1_1serving_1_1EventBus_1_1Subscription}{Subscription}} is not an owner of \mbox{\hyperlink{classtensorflow_1_1serving_1_1EventBus}{Event\+Bus}}.
