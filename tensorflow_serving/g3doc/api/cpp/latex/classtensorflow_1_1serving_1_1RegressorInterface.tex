\doxysubsection{Detailed Description}
Model agnostic interface for performing regression.

Specific implementations will exist for different model types (e.\+g. Tensor\+Flow Saved\+Model) that can convert the request into a model specific input and know how to convert the output into a generic Regression\+Result.
