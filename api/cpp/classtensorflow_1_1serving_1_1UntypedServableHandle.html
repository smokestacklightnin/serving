Detailed Description

A non-templatized handle to a servable, used internally in the Manager to retrieve a type-erased servable object from the Loader. The handle keeps the underlying object alive as long as the handle is alive. The frontend should not hold onto it for a long time, because holding it can delay servable reloading.
